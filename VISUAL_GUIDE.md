## Real Data - What to Look For

### ✅ Good Signs (Real Data)
- WPM between 80-180
- Filler count 0-20 (not 0, not 1000)
- Emotion percentages add to 100%
- Timestamps increase chronologically
- Duration matches interview length
- Performance scores between 1-10
- Video file actually created on disk

### ❌ Warning Signs (Fake Data)
- WPM of 1 or 0 (didn't speak)
- WPM over 500 (impossible)
- Emotion percentages don't add up
- Negative timestamps
- Score is 11/10 or -5/10
- Video file has size 0
